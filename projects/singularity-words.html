Приложение для изучения слов

1

Забыть через неделю

Студенты учат слова, сдают тест — и забывают 80% через неделю. Словарный запас не растёт. Преподаватели не видят, какие слова застряли.

Singularity Words борется с кривой забывания. Алгоритм подбирает слова, определяет момент повторения, адаптирует сложность. Целевая метрика: 1 слово в минуту с удержанием в долгосрочной памяти.

2

«Гармошка» — механика тренировки

Три версии механики. Первая — список слов. Вторая — интерактивные карточки. Третья — аккордеон: перевод, примеры, синонимы раскрываются постепенно.

Каждая итерация проверяла гипотезу. Список — хватит ли простого повторения? Карточки — помогает ли интерактив? Гармошка — улучшает ли контекст запоминание? A/B-тесты подтвердили: гармошка даёт лучший результат.

Слово считается «отработанным», если переведено верно в 60%+ попыток. Три режима: чтение, произнесение вслух, написание.

Drilling v.3 «Гармошка»: перевод, примеры, три режима тренировки
iOS: нативная реализация drilling с SF Symbols и системными контролами
Speaking v.3: голосовой режим — распознавание речи, повторная запись, подсказки
Drilling для N-грамм: тренировка фраз и словосочетаний, итерации механики
3

Интервальные повторения

Алгоритм определяет момент, когда слово вот-вот забудется, и возвращает его в тренировку. Слово начинается скрытым: r_____y. Буквы появляются по одной — если студент не вспомнил сам.

После тренировки — контекстный результат: «80% правильно, лучше чем 92% студентов». Соревновательный элемент без давления. Дизайн-решение: кнопку «Пропустить» переименовали в «Продолжить» — стало понятнее, что произойдёт.

Spaced Repetition: скрытое слово → подсказки по буквам → результат с процентилем
Редизайн Spaced Repetition (сентябрь 2024): обновлённая визуальная система повторений
4

Статистика и уровни

Уровни по CEFR: A1 → A2 → B1 → B2 с числовой точностью. Пример: уровень A2.32, допуск к сессии — B1.54+. Цель: 2 000 слов к сессии.

Активность по цветам: зелёный (80–100%), жёлтый (60–79%), оранжевый (40–59%), красный (0–39%). Тренды сравнивают текущую неделю с прошедшей. iOS-виджеты выносят прогресс на домашний экран.

iOS: уровни CEFR, недельный календарь, статистика по словам, лидерборд
5

Android + iOS

Одна UX-логика, две нативных реализации. Android — Material Design 3, XML-палитра, тёмная тема. iOS — SF Symbols, нативные контролы, WidgetKit-виджеты для рабочего стола.

Фид прошёл три версии. v.1 — табы (Слова / Грамматика / Профиль). v.2 — светлая тема, карточки тренировок с временными окнами. v.3 — нижняя навигация убрана, профиль переехал в top bar, Lottie-конфетти при завершении. Каждая итерация тестировалась на реальных студентах.

Эволюция фида: v1 табы → v2 светлые карточки с временными окнами → v3 профиль в top bar, Lottie-конфетти
Android: Cards v.2 Light, единый фид с дневной сессией
iOS: WidgetKit-виджеты, трекинг активности, прогресс по словам и грамматике
Итерации iOS-виджетов: статистика v1.2 → single widget → double widget (слова + грамматика)
