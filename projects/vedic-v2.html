01 — Нулевой рынок

Рынок есть. Продукта — нет.

Рынок астрологии в Турции — $5M+. Одно приложение Faladdin набрало 20 миллионов загрузок. Ведическая астрология в Турции: ноль цифровых продуктов. 2–3 практика, все работают офлайн.

Dr. Sugata — PhD из Amrita University, 17 лет практики, автор книги «Karma ve Karmanın Efendisi: Saturn», преподаватель санскрита. Единственный эксперт с академическим бэкграундом в регионе. Нужна платформа.

02 — Не гадание — образование

«Показывает тенденцию», не «произойдёт»

2025: основателя Faladdin задержали. «Fal» (гадание) — юридический риск в Турции. Решение по позиционированию: образование и самопознание, не предсказания.

Классические тексты (BPHS, Phaladeepika) как источник, не интуиция. Платформа-бренд, не персональный консалтинг. Консультации, курсы, сообщество — вынесены на будущий личный сайт эксперта.

Sade Sati: три фазы транзита Сатурна, образовательный контент, glass-карточки
03 — Калькулятор натальной карты

Три API, один шаг за раз

Пошаговая форма: дата → время → город. Три API: VedAstro для расчётов, OpenStreetMap для геокодинга, TimeAPI для часовых поясов.

AI интерпретирует карту. Бесплатный уровень: Groq (Llama 3.3 70B). Платные отчёты: только Claude — качество должно соответствовать цене. Калькулятор запущен в бете. Эксперт проверила свою карту, нашла ошибки. Точность важнее скорости запуска.

Калькулятор: пошаговая форма, бета-баннер, seed of life на фоне
04 — Три уровня дохода

Бесплатное остаётся бесплатным

Бесплатный уровень: калькулятор, блог, глоссарий из 50+ терминов. Цель — SEO-трафик и email-сбор. Нулевая конкуренция по турецким ведическим запросам.

Платные отчёты: Sade Sati (транзит Сатурна), натальная карта, бандл. 799–1 499 TRY с рассрочкой через Shopier. Taksit (рассрочка) — культурная норма в Турции.

Будущее: подписка «VedikCircle», 8-недельный курс, персональные консультации. Каждый уровень использует свою AI-модель. Премиум платит за качество.

Отчёты: Sade Sati 799₺, натальная карта 999₺, бандл 1 499₺. Glassmorphism-карточки
05 — Визуальный язык

Космос, не мистицизм

Тёмная космическая тема. Анимированное звёздное поле, 60 звёзд. Glass-морфизм карточки. Сакральная геометрия — текстура, не декорация.

Типографика: New York (антиква) для заголовков, Oceanic Grotesk (гротеск) для текста. Фиолетовый акцент (#a78bfa) — чакра Аджна. Золотой градиент на CTA. Пространство по золотому сечению (φ = 1.618).

Бриф клиента: «мягкие тона, медитативность, не мистицизм». Тёмная тема — первая версия. Светлая с пудровой палитрой — в плане.
