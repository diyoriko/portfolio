3

Калькулятор натальной карты

Пошаговая форма: дата → время → город. Три API: VedAstro для расчётов, OpenStreetMap для геокодинга, TimeAPI для часовых поясов.

AI интерпретирует карту. Бесплатный уровень: Groq (Llama 3.3 70B). Платные отчёты: только Claude — качество должно соответствовать цене. Калькулятор запущен в бете. Эксперт проверила свою карту, нашла ошибки. Точность важнее скорости запуска.

Калькулятор: пошаговая форма, бета-баннер

Три уровня дохода

Бесплатный уровень: калькулятор, блог, глоссарий из 50+ терминов. Цель — SEO-трафик и email-сбор. Нулевая конкуренция по турецким ведическим запросам.

Платные отчёты: Sade Sati (транзит Сатурна), натальная карта, бандл. 799–1 499 TRY с рассрочкой через Shopier. Taksit (рассрочка) — культурная норма в Турции.

Будущее: подписка «VedikCircle», 8-недельный курс, персональные консультации. Каждый уровень использует свою AI-модель. Бесплатное остаётся бесплатным. Премиум платит за качество.

Отчёты: Sade Sati 799₺, натальная карта 999₺, бандл 1 499₺
